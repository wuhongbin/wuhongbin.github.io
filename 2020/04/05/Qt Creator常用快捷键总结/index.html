Qt Creator 常用快捷键

快捷键 介绍
F1 查看帮助文档
Shift + F2 函数的声明和定义之间快速切换
F4 对应.h 文件和对应.cpp 文件切换
Ctrl + / 代码注释及取消注释（前提是选中代码）
Ctrl + 鼠标滚轮向上向下 调整字体变大变小
ctrl + shift + 键盘方向键 移动代码（前提是选中所要移动的代码）
ctrl + f 查找替换关键字
Ctrl + B 只编译（Build）不运行
Ctrl + R 编译并运行（run）
Ctrl + I 自动缩进当前行
Ctrl +M 创建书签
Ctrl + . 切换书签
Alt + M 打开书签栏
Ctrl + [ 光标跳到程序段开头
Ctrl + ] 光标跳到程序段结尾
